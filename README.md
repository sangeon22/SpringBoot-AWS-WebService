# Spring Boot, AWS 웹 서비스 
- OAuth 2.0 회원 등록 기능
- 게시판 CRUD 기능
- JUnit 테스트
- AWS EC2 운영 환경 및 AWS RDS 연동
- CI/CD 파이프라인
- 무중단 배포 가능
<br>

## 기술 스택 및 환경
- IntelliJ
- Java 8
- Gradle
- Spring Boot
- Spring Security
- Spring Data JPA
- JUnit
- JPA
- Mustache
- OAuth 2.0(Naver, Google)
- AWS (EC2, RDS, S3, CodeDeploy)
<br>

[배포자동화, 무중단 배포 과정은 아직 진행 중]
- Travis CI (유료화로 인하여 오픈 소스 CI Tool로 GitHub Action 진행예정)
- Nginx
<br>
